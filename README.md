## 功能
  本地mp3播放，显示网络歌词以及封面
  （由于歌词来自网络，歌曲市场不唯一，有可能出现歌词存在延迟现象）
  
## 使用
  修改***歌名格式为***： **歌名_歌手**.mp3
  
  支持拖拽文件(夹)来播放歌曲
  
  ![drag](https://github.com/user-attachments/assets/41427cb3-033b-482b-ad43-783121d3f08c)

  支持空格，小键盘左右键来控制歌曲播放状态。

  如需清除歌词缓存，点击如下选项。
  
  <img width="852" height="700" alt="image" src="https://github.com/user-attachments/assets/67979ccc-f5a6-4a8a-bb5e-c21025fb240e" />
